# Priorado Cavaleiros do Santo Graal nº 146 - Sergipe
🏛️ Priorado Cavaleiros do Santo Graal nº 146
Este é o repositório do website oficial do Priorado 146, localizado em Sergipe. O projeto foi desenvolvido para servir como um portal informativo e histórico, unindo a tradição da cavalaria com uma interface digital moderna e elegante.

🛡️ Sobre o Projeto
O site apresenta a trajetória de excelência e liderança do Priorado, consolidada desde sua fundação em 2010. Ele funciona como uma central de informações para membros e interessados, destacando:
História Institucional: Registro do legado de dedicação e conquistas da Ordem.
Galeria de Honoríficos: Exposição dos graus e honrarias de Comendadores e Grão-Cruzes.
Agenda de Reuniões: Sistema informativo dinâmico sobre encontros mensais.

✨ Destaques Visuais
O design foi construído seguindo uma estética nobre e solene:
Paleta de Cores: Uso predominante de Verde Musgo e Ouro Imperial.
Glassmorphism: Barra de navegação com efeito de vidro e desfoque de fundo
Interatividade: Botão flutuante pulsante para acesso rápido a informações de reuniões.
Responsividade: Interface adaptável para dispositivos móveis e desktops.

🚀 Tecnologias Utilizadas
HTML5: Estrutura semântica do conteúdo.
CSS3: Estilização avançada com gradientes, sombras (glow) e filtros.
JavaScript: Lógica de interação para alertas e controle de modais.
Bootstrap 5.3: Framework para agilidade no layout e responsividade.

📅 Informações de Reunião
As informações logísticas integradas ao site são:
Frequência: Último sábado de cada mês.
Horário: 14:00 horas.
Local: Definido pela diretoria.
